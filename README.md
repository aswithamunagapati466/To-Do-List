# To-Do-List
A responsive To-Do List application built using HTML, CSS, and JavaScript. It supports full CRUD operations, task filtering, completion tracking, and local storage to persist user data across browser sessions.

## 🔗 Live Demo
https://aswithamunagapati466.github.io/To-Do-List/

## 📂 GitHub Repository
https://github.com/aswithamunagapati466/To-Do-List

## ✨ Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks (All, Active, Completed)
- Save tasks using Local Storage
- Dynamic DOM Manipulation
- Responsive Design
- Task Counter

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- DOM Manipulation
- Local Storage API

## 🎯 Project Outcome

This project demonstrates JavaScript CRUD operations, DOM manipulation, event handling, filtering, and Local Storage by building a fully functional To-Do List application that retains user data across browser sessions.
